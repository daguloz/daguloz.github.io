Teoría de arrays

  • Distintas formas de crear un array

    Podemos usar el constructor new Array:

    var array = new Array();

    Al que podemos pasar un parámetro con el tamaño:

    var array = new Array(5);

    La opción más recomendada para crear un array es la siguiente:

    var array = [];
